# react-bigdatamx-search

## Install

`npm install react-bigdatamx-search`
-or-
`yarn add react-bigdatamx-search`

## Usage

```jsx
import BDMXSearchBar from 'react-bigdatamx-search';

// in your component
return (
  <BDMXSearchBar
    placeholder='This is optional'
  />
);
```
